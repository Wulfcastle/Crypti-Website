Current Release: Version v0.1.9с

This version focused mainly on performance enhancements and code optimization. All of the base code was completely re-factored and written to provide lower system requirements and much faster performance.

The blockchain loading times went from 2 hours to 10 mins. We also have optimized the client to not just work, but maintain performance on a 512mb cloud server.

For more information, read our official v0.1.9с release notes.
